Why Online Safety Matters

Many people use online resources to learn about birds, identify species, and report sightings. While these websites are helpful, they can also collect information about your device and your browsing activity. Understanding how websites work and how to protect your personal information can help you stay safe while exploring bird-related content online. This website is an experimental project created by a student to practice web development skills, including HTML and JavaScript. Native Missouri birds were chosen as the overall theme, while the main goal of the site is to demonstrate and apply different coding concepts.

What Websites Can Learn About Your Device

Websites can collect a wide range of information about your browser, device, and screen without you directly entering personal details. This information is accessed using built-in JavaScript objects such as the navigator and screen objects. The navigator object provides details about the user’s browser, language, and operating system, while the screen object provides information about the size and display of the user’s device. Websites use this information to improve functionality, adjust layouts for different screen sizes, and ensure compatibility across devices. Some examples of these properties include:

  • navigator.appName - Returns the name of the browser
  • navigator.appVersion - Returns version information about the browser
  • navigator.language - Returns the browser’s primary language
  • navigator.platform - Returns information about the user’s device platform
  • screen.width - Returns the total width of the screen in pixels
  • screen.height - Returns the total height of the screen in pixels

Tips for Staying Safe Online

  • Use strong and unique passwords for your accounts
  • Check for HTTPS before entering personal information
  • Avoid clicking on suspicious or unknown links
  • Keep your browser and device updated
  • Do not share personal information on untrusted websites
